# Homebrew Tap for Crowe Logic CLI

This is the official Homebrew tap for installing Crowe Logic CLI on macOS.

## Installation

```bash
# Add the tap
brew tap michaelcrowe/crowelogic https://github.com/michaelcrowe/homebrew-crowelogic

# Install the CLI
brew install crowelogic
```

## Upgrade

```bash
brew update && brew upgrade crowelogic
```

## Uninstall

```bash
brew uninstall crowelogic
brew untap michaelcrowe/crowelogic
```

## Development

To install from HEAD (latest main branch):

```bash
brew install --HEAD crowelogic
```

## Troubleshooting

If you encounter issues:

```bash
# Check installation
brew doctor

# Reinstall
brew reinstall crowelogic

# View logs
brew log crowelogic
```
